# ALT://Rundown 1.0: Deviation

## Overview

- **Release Date:** October 20, 2022
- **Tiers:** A, B, C, D
- **Total Expeditions:** 6
- **Location:** Mines of the Complex

## Narrative Summary

ALT://Rundown 1.0 is a re-release of the original Rundown 001 following the rundown system rework. Prisoners are tasked with progressively gaining security clearance by retrieving DNA samples and personnel IDs to access deeper sections of the Complex. The rundown culminates in shutting down the power grid in the D-tier, where The Warden and shadow-type enemies are introduced.

## New Features & Content

- **New Enemies:** Chargers (C-tier), Shadow enemies (D-tier)
- **Restored Weapons:** Combat Shotgun (Buckland AF6), Techman Arbalist V Machine Gun
- **Key Mechanic:** Dispersed encryption matrix requiring partial decoders
