# R1A1 - "The Admin"

## Overview

- **Rundown:** ALT://Rundown 1.0: Deviation
- **Tier:** A
- **Environment:** Storage (Main sector)
- **Dimensions:** No

## Objectives

### Main Objective

**Retrieve HSU:** Navigate to Zone 50 and acquire a key, unlock the path to Zone 52, locate and scan the correct HSU unit, then extract. The HSU scan triggers an Error Alarm that spawns 3 Strikers approximately every 8 seconds until extraction.

### Secondary Objective

None (Zone 51 is an optional resource collection zone with no sleeper enemies or alarms)

### Overload Objective

None

## Enemy Types

- Striker
- Shooter
- Giant
- Big Shooter
- Scout (2 present in Zone 50)

## Level Description

R1A1 is the first expedition in ALT://Rundown 1.0 and serves as the introductory/tutorial level for GTFO. It teaches fundamental gameplay mechanics including stealth, security doors, alarms, keys, and common enemy encounters. Players progress through zones, dealing with sleeping enemies and alarm-triggered waves, culminating in an HSU scan that triggers an Error Alarm requiring a fighting retreat to extraction.

The level introduces sleeping Giants for the first time. Zone 51 exists as an optional side area for resource gathering with no enemies.

## Notable Mechanics

- **Class II Alarm** in Zone 50
- **Class IV Alarm** in Zone 52
- **Checkpoint** located in Zone 52
- **Error Alarm** triggers after HSU scan completion (3 Strikers every ~8 seconds)
- **Tutorial level** introducing core mechanics: stealth, security doors, alarms, keys
