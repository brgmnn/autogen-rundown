# R1B2 - "The Officer"

## Overview

- **Rundown:** ALT://Rundown 1.0: Deviation
- **Tier:** B
- **Environment:** Refinery (Main sector)
- **Dimensions:** No

## Objectives

### Main Objective

**Retrieve HSU:** Use QUERY terminal to determine HSU location (Zone 20 or 21), navigate through alarm doors to obtain an HSU access key, then scan the HSU and extract. The HSU scan triggers an Error Alarm spawning 3 Strikers every ~8 seconds.

Steps:

1. Use QUERY to determine HSU location in Zone 20 or 21
2. Enter Zone 17 (contains Scout)
3. Enter Zone 18 (Class II Alarm); optionally collect colored key for Zone 19
4. Enter Zone 22 or 23 (Class III Alarm) to obtain HSU access key
5. Enter Zone 20 or 21 (Class IV Alarm) to scan HSU; triggers Error Alarm
6. Retreat and extract

### Secondary Objective

None

### Overload Objective

None

## Enemy Types

- Shooter
- Big Shooter
- Striker (alarm/Scout waves only)
- Giant (alarm/Scout waves only)
- Scout

## Level Description

R1B2 is distinctive because every sleeping enemy in the level is either a Shooter or a Big Shooter. Strikers and Giants only appear during alarm-triggered waves or Scout detection waves, never as sleeping enemies. This creates a unique combat dynamic where stealth encounters are exclusively ranged-enemy focused.

The level has limited ammo availability throughout and only 1 closable door for alarm defense, making resource management and positioning critical. Zone 19, accessible via an optional colored key, contains significantly higher resources (140% ammo vs the standard 40%).
