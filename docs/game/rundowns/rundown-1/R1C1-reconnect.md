# R1C1 - "Reconnect"

## Overview

- **Rundown:** ALT://Rundown 1.0: Deviation
- **Tier:** C
- **Environment:** Mines (Main sector)
- **Dimensions:** No

## Objectives

### Main Objective

**Reactor Startup:** A wave-based, combat-focused mission where large hordes of enemies assault the players in timed intervals. Navigate through zones to Zone 67, initiate the reactor startup, survive 8 reactor phases with escalating enemy waves, then extract.

Steps:

1. Enter Zone 63 via Class I Stealth Scan, proceed through successive zones to Zone 67
2. Initiate Reactor Startup and survive 8 reactor phases
3. After completing the reactor sequence, an Error Alarm spawns enemies
4. Retreat to entry point and extract

### Secondary Objective

None

### Overload Objective

None

## Enemy Types

- Striker
- Shooter
- Giant
- Big Shooter
- Hybrid (highly dangerous enemy that fires a stream of homing projectiles; introduced in Phase 4)

## Level Description

R1C1 is a combat-intensive reactor startup mission set in the Mines environment. The level begins with all lights OFF; lights activate only when the reactor startup is initiated. Players must navigate through zones using Stealth Security Scans (non-alarming door scans) before reaching the reactor in Zone 67.

The reactor startup consists of 8 phases with escalating difficulty. Prep times range from 20 seconds to 1 minute 50 seconds, with wave durations of 1-1.5 minutes. Code input time is 45 seconds throughout all phases. Phases 4, 6, and 8 introduce Hybrids alongside standard enemy types.

After completing the reactor sequence, an Error Alarm activates that spawns 2 Strikers/Shooters every ~8 seconds, scaling to 4 enemies per wave over one minute. Players must retreat to the entry point for extraction.

The Hybrid enemy was originally found in Rundown 002, making this expedition a reintroduction of that threat.

## Notable Mechanics

- **Reactor Startup** mission type with 8 phases of escalating waves
- **Stealth Security Scan** (Class I) - operates like a standard scan but does not trigger an alarm
- **No alarmed doors** in the level
- **All lights initially OFF** - lights activate upon reactor startup initiation
- **Hybrids** introduced in phases 4, 6, and 8 (homing projectiles)
- **Error Alarm** after reactor completion (2 enemies every ~8 seconds, scaling to 4)
- **Distinct reactor spawn mechanics** separate from standard alarm spawns
- **Optimal hold position:** Zone 65B (the bridge room) with enemies spawning close in Zone 64B
- **Recommended loadout:** Bio Tracker + 3 Sentry Guns (Burst and Sniper types preferred)
