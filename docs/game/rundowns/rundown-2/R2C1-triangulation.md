# R2C1 - "Triangulation"

## Overview

- **Rundown:** ALT://Rundown 2.0: Infection
- **Tier:** C
- **Environment:** Mines (Main sector)
- **Dimensions:** No

## Objectives

### Main Objective

Establish Uplink (x3): Complete three Terminal Uplinks in sequence across different zones. Each uplink is a 3-code uplink. After all three uplinks are completed, an Error Alarm activates that spawns 2 Shadows every ~10 seconds for the remainder of the expedition. Strategic sequencing is recommended, with the final uplink in Zone 202/203 to facilitate easier extraction.

Uplink locations:

1. Zone 202/203 - Requires QUERY to locate terminal (3-CODE UPLINK)
2. Zone 204 - Contains 3 Scouts; CLASS V MIXED ALARM (3-CODE UPLINK)
3. Zone 206/207 - Behind BLOOD DOOR, requires QUERY (3-CODE UPLINK)

### Secondary Objective

Retrieve the Fog Turbine from Zone 204 (recommended, as zones 204-207 are filled with thick fog)

### Overload Objective

None

## Enemy Types

- Striker
- Shooter
- Giant
- Big Shooter
- Scout
- Shadow (spawns exclusively from Terminal Uplink alarms and the Error Alarm)
- Spitter (infection-spreading)

## Level Description

Triangulation is the first C-tier expedition and expands on the Establish Uplink mission type introduced in R2B3. Instead of a single uplink, players must complete three separate 3-code uplinks across different zones, each with its own challenges and alarm defenses.

The level features thick fog in zones 204-207, making the optional Fog Turbine from Zone 204 highly recommended. The Shadow enemy type returns in this expedition, spawning exclusively from uplink-related alarms and the post-completion Error Alarm. This creates a unique combat dynamic where standard enemies populate the zones but Shadows are the primary wave threat.

After completing all three uplinks, a persistent Error Alarm spawns 2 Shadows every ~10 seconds, creating significant time pressure for extraction. Strategic planning of uplink order is crucial -- completing the final uplink closest to extraction is recommended.

## Notable Mechanics

- **Triple uplink sequence:** Three separate 3-code uplinks must be completed across different zones
- **Shadow-exclusive alarm spawns:** Terminal uplink alarms and the Error Alarm only spawn Shadow enemies
- **Persistent Error Alarm:** After all uplinks complete, spawns 2 Shadows every ~10 seconds
- **Thick fog:** Zones 204-207 filled with fog; Fog Turbine available in Zone 204
- **Blood Door:** Access to Zones 206/207
- **CLASS V MIXED ALARM:** Zone 204 features a mixed alarm type
- **CLASS III ALARM:** Zone 205 alarm spawns only Shadows
- **Zones:** 201, 202, 203, 204, 205, 206, 207 with varying resource distributions; Zone 204 has highest ammo (140) and medical (200)
