# R3A3 - "Bolt"

## Overview

- **Rundown:** ALT://Rundown 3.0: The Vessel
- **Tier:** A
- **Environment:** Mines
- **Zones:** 7 (Zone 428-434)
- **Dimensions:** No

## Objectives

### Main Objective

**Reactor Startup:** Enter Zone 430 and start the Reactor Sequence. Survive all six reactor phases across multiple zones, then return to extract. Each phase requires entering verification codes at terminals in specific zones within a time limit while surviving enemy waves.

### Secondary Objective

None

### Overload Objective

None

## Level Description

R3A3 serves as an easier version of R2D2, similar to how R3A2 relates to R2D1. The level centers around a six-phase reactor startup sequence with escalating difficulty. Doors to Zones 431-434 only open when their respective objectives activate, meaning players must react quickly to reach the correct terminal locations.

The reactor phases are spread across multiple zones, requiring coordination to locate and input codes at various terminals while defending against enemy waves. Each phase has specific code entry times, preparation periods, and wave durations that escalate in intensity.

The final wave ends with a large group of active Giants that must be avoided or carefully managed, providing a tense conclusion before extraction.

## Notable Mechanics

- **Reactor Sequence** with 6 phases across multiple zones
- **CLASS III ALARM** triggered during Phase 5
- Doors to Zones 431-434 only unlock when their respective phase activates
- Multiple terminal locations per zone (2-3 options per phase)
- Final wave features a large group of active Giants
