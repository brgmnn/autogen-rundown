# R4A2 - "Foster"

## Overview

- **Rundown:** ALT://Rundown 4.0: Contact
- **Tier:** A
- **Environment:** Mines
- **Dimensions:** No

## Objectives

### Main Objective

Retrieve Item (Cryogenic Cases): Retrieve two Cryogenic Cases from designated zones and extract them. Navigate to Zone 44 (Class III Alarm) to collect the first case and a Power Cell, then access Zone 43 where an optional Bulkhead Key and additional Power Cell can be found. Use a Power Cell to enter Zone 45/46 (Class IV Alarm) for the second case before returning for extraction.

### Secondary Objective

Retrieve HSU: Unlock Zone 108 with a Bulkhead Key, use a QUERY command to locate an HSU in Zones 110/111/112, and complete a team scan (Class I Alarm). Players must manage Power Cells carefully, as extra cells exist but resource scarcity is possible.

### Overload Objective

Terminal Command: Use the Bulkhead Key at Zone 108 to access Zone 241, then input the command "DISABLE_LIFE_SUPPORT" into a terminal. Opening this door initiates an Error Alarm spawning 2 Strikers/Shooters every ~15 seconds for the expedition's duration.

## Enemy Types

- Striker
- Shooter
- Giant
- Big Shooter
- Scout

## Level Description

R4A2 takes place in the Mines environment and features all three sector types: Main, Secondary, and Overload. This is the first instance where an optional sector (the Overload) directly affects the Main objective through its Error Alarm mechanic. The Overload introduces the Terminal Command objective type, requiring players to input specific commands at designated terminals.

## Notable Mechanics

- **Prisoner Efficiency:** Completing all three objectives in a single run requires completing the Overload before the Secondary.
- **Error Alarm:** Completing the Overload objective triggers continuous enemy spawning (2 Strikers/Shooters every ~15 seconds) for the remainder of the expedition.
- **Terminal Command:** New objective type introduced here, requiring players to input specific commands into terminals.
- **Optional Sector Impact:** First instance where an optional sector directly impacts the Main objective experience.
- **Zones:** 42, 43, 44, 45, 46 (Main), 108, 109, 110, 111, 112 (Secondary), 241 (Overload).
