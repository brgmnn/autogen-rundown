# R4B1 - "Malachite"

## Overview

- **Rundown:** ALT://Rundown 4.0: Contact
- **Tier:** B
- **Environment:** Research
- **Dimensions:** No

## Objectives

### Main Objective

Distribute Powercells: Locate and insert three Power Cells into Generators across Zones 51, 52/53, and 54. Collect required keys and navigate to extraction in Zone 56. Optional: Retrieve a Fog Turbine in Zone 51. Access Zone 57 with an extra Power Cell for bonus resources and a Disinfection Station. Zone 55 features a CLASS III ALARM, Zone 56 features a CLASS IV ALARM.

### Secondary Objective

Reactor Startup: Access Zone 221 using the extra Power Cell. Complete a five-phase reactor sequence with code inputs at designated terminals in Zones 222, 223, and 225. Collect a Bulkhead Key from Zone 227. Attempting this objective prevents access to Zone 57 (bonus resource room). Zone 227 contains a sleeping Mother enemy (killing not required). Zones 224/226 feature CLASS III CLUSTER ALARMs, Zone 225 has a BLOOD DOOR.

**Reactor Phases:**

- Phase 1: 45-second code input, 30-second prep, 90-second waves
- Phase 2: 4-minute code window at Zone 222
- Phase 3: 5-minute code window at Zone 223 (2 terminals)
- Phase 4: 8-minute code window at Zone 225 (2 terminals)
- Phase 5: 9-minute code window at Zone 227 (1 terminal)

### Overload Objective

None

## Level Description

The expedition title refers to the Malachite gemstone, a greenish/teal mineral which corresponds with the green infectious fog that permeates the level. This is a Research environment expedition characterized by its fog-heavy atmosphere and the strategic choice between accessing the bonus resource room (Zone 57) or pursuing the Secondary reactor startup objective.

Zone 57 is notable for being a resource room only accessible during a main-only run, containing 220 Ammo, 120 Medi, and 120 Tool, plus a Disinfection Station. Players must choose between this bonus or the Secondary objective since both require the same Power Cell.

## Notable Mechanics

- **Infectious Fog:** Covers the entire expedition, requiring careful resource management and use of disinfection items.
- **Fog Turbine:** Optional pickup in Zone 51 to help manage fog.
- **Disinfection Station:** Available in Zone 57 (bonus room).
- **Strategic Resource Choice:** Players must choose between Zone 57's resources/disinfection or the Secondary objective, as both require the same Power Cell.
- **Reactor Startup:** Five-phase sequence with escalating code windows and terminal locations across multiple zones.
- **Blood Door:** Zone 225 during the Secondary objective.
- **Alarm Doors:** CLASS III (Zone 55), CLASS IV (Zone 56), CLASS III CLUSTER (Zones 224/226).
- **Sleeping Mother:** Zone 227 contains a Mother boss that does not need to be killed.
