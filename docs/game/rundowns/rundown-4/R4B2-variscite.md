# R4B2 - "Variscite"

## Overview

- **Rundown:** ALT://Rundown 4.0: Contact
- **Tier:** B
- **Environment:** Mines
- **Dimensions:** No

## Objectives

### Main Objective

Activate Generator Cluster: Collect a Bulkhead Key from Zone 84 and use it to access Zone 85 containing a Generator Cluster. Retrieve four Power Cells from Zones 86-89. Insert cells into the cluster, which progressively lowers fog. Inserting the final cell triggers a Class I Alarm and enables extraction from Zone 84.

### Secondary Objective

Terminal Command: Access Zone 177 via Bulkhead Door in Zone 84 (initiates Error Alarm spawning enemies every ~50 seconds). Enter command "SCCS_LOCAL_TEMP_SET_294_GRAD_INV_FALSE" into a terminal. The zone contains two Scouts. Collect one or both Bulkhead Keys before returning to complete the main objective.

### Overload Objective

Gather Items (OSIP Hormones): Access Zone 291 via Door Controller in Zone 87. Collect 6 OSIP Hormones from available zones (9 total can be found). Three of four optional zones must be explored: Zone 292 (Class VI Alarm, three Scouts, abundant resources), Zone 293 (Class IV Alarm, Blood Door), Zone 294 (contains Scout), or Zone 295 (Class V Alarm, two sleeping Mothers). Return to main area upon completion.

## Level Description

R4B2 is notable for being the first expedition to feature standard (non-infectious) fog in an Activate Generator Cluster mission. Unlike the usual infectious fog in such missions, this level uses standard fog that interacts dynamically with Power Cell placement -- some areas lose fog while others gain it, requiring strategic consideration of cell insertion order.

The expedition takes place in the Mines environment and features all three sector types. The Secondary sector introduces a persistent Error Alarm upon entry, and the Overload sector contains some of the highest alarm classes in the B-tier along with Mother boss encounters.

## Notable Mechanics

- **Dynamic Fog System:** First Activate Generator Cluster mission with standard (non-infectious) fog. Placing Power Cells has variable effects -- some areas lose fog while others gain it.
- **Error Alarm:** Opening the Secondary Bulkhead Door initiates a persistent Error Alarm spawning enemies throughout the expedition (~50 second intervals).
- **Prisoner Efficiency:** Secondary zone must be entered before Overload objective to achieve all three objectives in one run.
- **High-Class Alarms:** Overload features Class IV, V, and VI Alarms.
- **Blood Door:** Zone 293 in the Overload sector.
- **Sleeping Mothers:** Zone 295 contains two sleeping Mothers.
- **Resource Distribution:** Zone 292 (Overload) provides exceptional resources (140% Ammo, 100% Medi, 40% Tool).
