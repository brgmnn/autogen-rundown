# R4B3 - "Chrysolite"

## Overview

- **Rundown:** ALT://Rundown 4.0: Contact
- **Tier:** B
- **Environment:** Research
- **Dimensions:** No

## Objectives

### Main Objective

Establish Uplink: Explore Zone 37 to collect a Bulkhead Key and Colored Key, then establish three 4-CODE UPLINKs across Zones 38, 39, and 40/41. Zone 38 features a CLASS II CLUSTER ALARM. Zone 40/41 has a CLASS III ALARM. Zone 41 requires inserting a Power Cell into a Generator. Completing all uplinks triggers an Extraction Alarm spawning 3 Strikers/Shooters every ~6 seconds.

### Secondary Objective

Retrieve Item (Cryogenic Cases): Access Zone 123 via Bulkhead Door using the key from Zone 37. Progress through Zones 123-124 (each containing 2 Scouts), then enter Zone 125 (CLASS III ALARM) and Zone 126 (CLASS IV ALARM) to collect two Cryogenic Cases. These zones contain large Giant concentrations. Return both cases to Zone 37 for extraction.

### Overload Objective

Terminal Command: Unlock access to Zone 363 via Bulkhead Door in Zone 124. Zone 363 (CLASS I LARGE SUSTAIN ALARM) contains 2 Scouts and concentrated Giants/Big Shooters. Enter Zone 364 (CLASS IV CLUSTER ALARM) or Zone 365 (CLASS V CLUSTER ALARM) to obtain a Colored Key. Zone 366 (CLASS I LARGE SUSTAIN ALARM with BLOOD DOOR) requires executing command "DEACTIVATE_FILTERS_S65T" and features an active Mother boss alongside Hybrids.

## Level Description

The expedition title references the Chrysolite gemstone, following the B-tier gemstone naming convention. R4B3 is a Research environment level that introduces the Sustain Alarm mechanic -- a new alarm variant that reverses scan progress if team members are downed or leave the scan area. These are typically large versions requiring 1+ minute completion times, forcing defensive positioning.

The level features all three sectors with escalating difficulty. The Main objective's three uplinks can be completed in any order, offering tactical flexibility. The Overload sector features the most dangerous encounters, including an active Mother boss in Zone 366.

## Notable Mechanics

- **Sustain Alarm (New):** Introduced in this expedition. Reverses scan progress if team members are downed or leave the scan area. Large versions require 1+ minute to complete, forcing defensive positioning.
- **Extraction Alarm:** Completing all three Main uplinks triggers an extraction alarm spawning 3 Strikers/Shooters every ~6 seconds.
- **Cluster Alarms:** CLASS II (Zone 38), CLASS IV (Zone 364), CLASS V (Zone 365).
- **Blood Door:** Zone 366 in the Overload sector.
- **Active Mother Boss:** Zone 366 features an active (not sleeping) Mother alongside Hybrids.
- **Uplink Flexibility:** The three main uplinks can be completed in any order.
- **Prisoner Efficiency:** Achievable by completing Secondary and Overload objectives before the Main objective. Recommended to complete the final uplink in Zone 38 or 39 last to ease extraction.
- **Resource Distribution:** Main zones (37-41) range 0-80% Ammo, 40-60% Medi, 0-40% Tool. Secondary zones (123-126) range 0-120% Ammo, 40-140% Medi, 0-60% Tool. Overload zones (363-366) range 80-120% Ammo, 60-120% Medi, 20-60% Tool.
