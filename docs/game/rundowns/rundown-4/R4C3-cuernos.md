# R4C3 - "Cuernos"

## Overview

- **Rundown:** ALT://Rundown 4.0: Contact
- **Tier:** C
- **Environment:** Mines
- **Dimensions:** No

## Objectives

### Main Objective

Retrieve Item (Fog Turbine): Obtain a Bulkhead Key in Zone 148. Retrieve a Colored Key from Zone 149 or 150 (CLASS III M ALARM). Progress to Zone 151. Use the Bulkhead Key to access Zone 152 (CLASS III M ALARM). Collect another Colored Key from Zone 154 or 155 to reach Zone 153. Extract the Fog Turbine from Zone 153 (CLASS IV MIXED ALARM). Return to Zone 148 entry point for extraction (triggers Extraction Scan alarm).

### Secondary Objective

Activate Generator Cluster: Use Bulkhead Key at Zone 151 to access Zone 264. Collect two Power Cells from Zones 265-266 (CLASS VI ALARM). Insert Power Cell 1 to lower fog. Insert Power Cell 2 to raise fog and eliminate infection (triggers CLASS IV ALARM). Retrieve Bulkhead Key from Zone 267 and return to main objective.

### Overload Objective

Retrieve HSU: Use Bulkhead Key in Zone 148 to open Zone 320. Collect Power Cell from Zone 320. Access HSU through Zone 321 or 322. Complete Team Scan at HSU (triggers Error Alarm spawning four Chargers approximately every 45 seconds for remainder of expedition). Retrieve Bulkhead Key from Zone 323 (BLOOD DOOR -- spawns Tank boss). Return to main objective.

## Level Description

R4C3 is the ninth expedition and is described as "characteristically Charger-focused," with Charger enemies comprising significant portions of both the Main and Overload objectives. The level takes place in the Mines environment with infectious fog as a persistent hazard.

The Secondary objective introduces an interesting fog manipulation mechanic where inserting Power Cells dynamically changes the fog state -- the first cell lowers fog while the second raises it but eliminates infection. The Overload objective features a guaranteed Tank boss encounter behind a Blood Door in Zone 323.

## Notable Mechanics

- **Charger-Focused Design:** Charger and Big Charger enemies are the defining combat challenge across Main and Overload sectors.
- **Dynamic Fog Manipulation:** Secondary objective Power Cells have different effects -- first lowers fog, second raises fog but eliminates infection.
- **Mixed Alarms:** CLASS III M and CLASS IV MIXED ALARM types used in the Main sector.
- **CLASS VI ALARM:** High-class alarm in the Secondary sector (Zones 265/266).
- **Error Alarm:** Overload HSU Team Scan triggers persistent Charger spawns (4 every ~45 seconds).
- **Blood Door:** Zone 323 spawns a Tank boss.
- **Infectious Fog:** Present in Main and Overload sectors. Spitters cause infection in Main.
- **Extraction Alarm:** Triggered when returning with the Fog Turbine.
- **Prisoner Efficiency:** Recommended completion order is Secondary, then Overload, then Main. Optional strategy: bringing an extra Power Cell from Overload allows skipping Zone 266.
