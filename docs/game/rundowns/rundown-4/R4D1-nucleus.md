# R4D1 - "Nucleus"

## Overview

- **Rundown:** ALT://Rundown 4.0: Contact
- **Tier:** D
- **Environment:** Mines
- **Dimensions:** No

## Objectives

### Main Objective

Reactor Startup: Reach the Reactor in Zone 62 and initiate the startup sequence. Complete four phases of startup codes at different zone terminals, then return to Zone 60 for extraction. An optional Bulkhead Key collection in Zone 67 is available after the third reactor wave.

**Reactor Phases:**

- Phase 1 (Zone 64): 5m 30s preparation, 2m 20s wave
- Phase 2 (Zone 66): 6m 30s preparation, 2m 50s wave
- Phase 3 (Zone 69): 7m preparation, 3m 40s wave
- Phase 4 (Zone 67): 10m preparation, 4m wave

### Secondary Objective

Terminal Command: Use a Bulkhead Key and the Bulkhead Door Controller in Zone 60 to open the Door to Zone 368. Traverse Zones 369-370, encountering Mother creatures and Giants. Enter command `SPFC_REACTOR_A42_LINE_7_TRUE` at a designated terminal in Zone 370, then extract.

## Level Description

R4D1 is the first D-tier expedition and takes place in the Mines environment. The level centers around a Reactor Startup sequence with four phases of escalating difficulty. Each phase requires finding and inputting codes at terminals scattered across different zones within time limits, while defending against enemy waves between phases.

The Secondary sector features encounters with Mother creatures and Giants, culminating in a terminal command input. The reactor phases feature increasingly longer preparation times and wave durations, with the final phase giving 10 minutes to find codes but requiring survival through a 4-minute enemy wave.

## Notable Mechanics

- **Reactor Startup:** Four-phase sequence with escalating time windows and wave durations. Code terminals are distributed across Zones 64, 66, 69, and 67.
- **Escalating Phases:** Preparation times increase from 5m 30s to 10m; wave durations increase from 2m 20s to 4m.
- **Multiple Terminal Locations:** Each phase may have multiple possible terminal locations requiring search.
- **Door Awakening:** Secondary objective zones feature door awakening mechanics.
- **Mother Encounters:** Secondary sector features Mother bosses with Baby Striker spawns.
- **Charger Variants:** Main sector includes both Charger and Big Charger enemies.
