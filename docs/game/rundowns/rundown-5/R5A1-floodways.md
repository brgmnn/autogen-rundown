# R5A1 - "Floodways"

## Overview

- **Rundown:** ALT://Rundown 5.0: Rebirth
- **Tier:** A
- **Environment:** Floodways (overgrown sewer system)
- **Dimensions:** No

## Objectives

### Main Objective

Terminal Command: Progress from Zone 61 through Zone 65, retrieving a Power Cell in Zone 62 and inserting it into a Generator in Zone 62E. Input the command "OVERRIDE_LOCKDOWN" at a terminal in Zone 63 (Class II Alarm). Return to Zone 61F, enter Zone 64 (a Cocoon respawn zone), then proceed to Zone 65 (Class III Cluster Alarm) and extract.

### Secondary Objective

Gather Items: Use a Bulkhead Key at Zone 64C to unlock Zone 191. Power a Generator in Zone 191 using a Power Cell. Enter Zones 192/193/194 to collect a minimum of 6 out of 7 available Plant Samples, then return and complete the main objective.

### Overload Objective

None

## Enemy Types

- Striker
- Shooter
- Giant
- Big Shooter
- Scout

## Level Description

Floodways is the first expedition of Rundown 5, set in an overgrown sewer system with notable amounts of overgrowth compared to other Complex locations. The level introduces the Cocoon mechanic, which causes enemies to respawn in Zone 64. The secondary objective involves collecting Plant Samples labeled with "Kovac Defense Services" (spelled as "Kovak" on the samples themselves). Early Access versions had Cocoons in most main zones, but these were later reduced to only Zone 64. This is also the first instance in the rundown series where logs appear outside the main sector.

## Notable Mechanics

- **Cocoons:** Introduced in this level; present in Zone 64 causing enemy respawns
- **Alarm Doors:** Class II Alarm (Zone 63), Class III Cluster Alarm (Zone 65), Class III Small Sustain Alarm
- **Bulkhead Doors:** Multiple bulkhead doors with controllers requiring keys
- **Terminal Commands:** OVERRIDE_LOCKDOWN command required for progression
- **Zone Layout:** Main sector spans Zones 61-65; Secondary sector spans Zones 191-194
- Giants never spawn sleeping naturally on this expedition
