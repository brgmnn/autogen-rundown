# R5A2 - "Recollect"

## Overview

- **Rundown:** ALT://Rundown 5.0: Rebirth
- **Tier:** A
- **Environment:** Datacenter
- **Dimensions:** No

## Objectives

### Main Objective

Retrieve Item: Collect a Bulkhead Key in Zone 15, open a Bulkhead Door to Zone 16 (Class II Cluster Alarm). Enter Zone 17 or 18 to retrieve a colored Key for the Security Door to Zone 19. Enter Zone 20 (Class III Cluster Alarm) and pick up the HISEC Cargo, which triggers an Error Alarm spawning Strikers every ~10 seconds. Extract from Zone 15 with the cargo.

### Secondary Objective

Terminal Command: Open a Bulkhead Door to Zone 131 using the Zone 15 key. Input the command "REROUTE_POWER" into a terminal, which disables lights, unlocks Zone 132, and spawns a Snatcher. Collect the Bulkhead Key in Zone 132.

### Overload Objective

Gather Items: Enter Zone 302 and collect an HDD. The zone contains a Mother boss and Baby Strikers that awaken upon entry.

## Enemy Types

- Striker
- Shooter
- Giant
- Hybrid
- Scout
- Snatcher (Secondary sector)
- Mother (Overload sector boss)
- Baby Striker (Overload sector)

## Level Description

Recollect is set in a Datacenter environment with foggy, hard-to-traverse terrain. The main objective involves retrieving HISEC Cargo from deep within the level, which triggers a persistent Error Alarm on pickup. The secondary objective plunges the level into darkness by rerouting power, introducing a Snatcher enemy. The overload objective features a Mother boss encounter in a contained zone.

## Notable Mechanics

- **Error Alarm:** Triggered upon picking up HISEC Cargo; spawns Strikers every ~10 seconds
- **Darkness:** Secondary objective disables all lights in affected zones
- **Alarm Doors:** Class II Cluster Alarm (Zone 16), Class III Cluster Alarm (Zone 20)
- **Blood Doors:** Present in the level
- **Boss Encounters:** Mother in Zone 302 (Overload), Snatcher in Zone 131 area (Secondary)
- **Zone Layout:** Main sector Zones 15-20; Secondary sector Zones 131-132; Overload sector Zone 302
- Long Range Flashlights recommended for darkened secondary objective areas
- C-Foam Launcher has limited utility due to few closable doors
