# R5A3 - "Mining"

## Overview

- **Rundown:** ALT://Rundown 5.0: Rebirth
- **Tier:** A
- **Environment:** Mines
- **Dimensions:** No

## Objectives

### Main Objective

Terminal Command: Access terminals across Zones 39-43 to extract a datacore. Locate passwords from Zone 40's logs, unlock terminals progressively, and reach Zone 44 for extraction. Requires a Bulkhead Key from Zone 40 to proceed through Zone 41 (Class III Cluster Alarm). Zone 44 features a Class I Sustain Diminished Alarm.

### Secondary Objective

Retrieve HSU: Navigate to Zone 124 to scan a specific HSU unit among multiple candidates. Collect an additional Bulkhead Key from Zone 124. Activating Zone 124's door triggers an Error Alarm spawning 3 Strikers/Shooters every ~50 seconds for the rest of the mission. Zone 123 features a Class V Alarm.

### Overload Objective

Activate Generator Cluster: Traverse fog-filled Zones 231-235 using a Fog Turbine to collect three Power Cells, then deliver them to a Generator Cluster in Zone 231. This unlocks Zone 233 and raises infectious fog levels throughout affected areas. Zone 232 is a Cocoon zone with respawning Scouts. Zone 235 features a Class V Alarm.

## Enemy Types

- Striker
- Shooter
- Giant
- Big Shooter
- Scout

## Level Description

Mining is set in the Mines environment with infectious fog as a primary hazard. The level features a multi-step terminal puzzle for the main objective requiring password discovery from logs. The secondary objective introduces a persistent Error Alarm that lasts the remainder of the mission. The overload objective takes place in heavily fogged zones requiring a Fog Turbine for safe traversal, with Cocoon respawn mechanics adding pressure.

## Notable Mechanics

- **Infectious Fog:** Present throughout, especially in Overload sector zones; requires Fog Turbine
- **Cocoons:** Zone 232 features respawning Scouts
- **Alarm Doors:** Class III Cluster (Zone 41), Class V (Zones 123, 235), Class I Sustain Diminished (Zone 44)
- **Error Alarm:** Triggered by opening Zone 124; spawns 3 Strikers/Shooters every ~50 seconds permanently
- **Terminal Puzzles:** Locked terminals requiring passwords found in Zone 40 logs
- **Disinfection Stations:** Available to counter infectious fog
- **Prisoner Efficiency:** Recommended order is Overload, then Secondary, then Main
- **Zone Layout:** Main Zones 39-44; Secondary Zones 123-124; Overload Zones 231-235
