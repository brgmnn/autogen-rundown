# R5B1 - "Smother"

## Overview

- **Rundown:** ALT://Rundown 5.0: Rebirth
- **Tier:** B
- **Environment:** Mines
- **Dimensions:** No

## Objectives

### Main Objective

Retrieve HSU: Collect a Colored Key from Zone 93 or 97 to unlock Zone 94. Optionally retrieve the Fog Turbine in Zone 94. Enter Zone 96 or 98 to find and scan the correct HSU unit. Opening Zone 96/98 triggers an Error Alarm spawning two Big Chargers every ~45 seconds. HSU scan completion unlocks Zone 99 and intensifies fog density. Return to Zone 92 for extraction.

### Secondary Objective

Distribute Powercells: Use a Bulkhead Key with the Bulkhead Door Controller in Zone 94 to access Zone 223. Collect 3 Power Cells and distribute them to 3 Generators in Zones 224, 225, and 226. Recommended to complete after the Overload objective.

### Overload Objective

Reactor Startup: Use a Bulkhead Key from Zone 99 with the Bulkhead DC in that zone. Reach the Reactor Terminal in Zone 455 and complete a 7-phase Reactor Startup Sequence. All phases feature 30-second code input windows. Waves escalate from ~70 seconds (Phases 1-3) to 80 seconds (Phases 4-7). Phase 4 introduces Chargers; Phase 7 removes Giants. Completing the reactor transforms fog to Infectious Fog.

## Level Description

Smother is set in the Mines with fog as a central mechanic. The main objective involves scanning an HSU unit while dealing with a persistent Error Alarm that spawns Big Chargers. The Overload objective features a demanding 7-phase Reactor Startup sequence that transforms the existing fog into infectious fog upon completion. The extraction phase spawns Chargers, adding final pressure. The level features branching paths with zones appearing as either of two options (93/97, 96/98).

## Notable Mechanics

- **Infectious Fog:** Overload completion transforms all fog to infectious; spreads between sectors
- **Error Alarm:** Triggered during HSU scan; spawns 2 Big Chargers every ~45 seconds
- **Extraction Alarm:** Spawns Chargers during final extraction
- **Reactor Startup:** 7-phase sequence with escalating wave difficulty
- **Branching Paths:** Zones vary between runs (93/97, 96/98)
- **Prisoner Efficiency Route:** Main, then Overload, then Secondary
- **Zone Layout:** Main Zones 92-99; Secondary Zones 223-226; Overload Zones 452-456
- Zone 99 is resource-rich (140% ammo, 180% medi, 160% tool)
