# R5B3 - "Unseal"

## Overview

- **Rundown:** ALT://Rundown 5.0: Rebirth
- **Tier:** B
- **Environment:** Mines
- **Dimensions:** No

## Objectives

### Main Objective

Process Item: Retrieve HISEC Cargo from Zone 45 and open the Security Door to Zone 46. Opening the door triggers an Error Alarm spawning 6 Chargers approximately every 2 minutes 40 seconds until the cargo is processed. Proceed to Zone 48 (Class IV Mixed Alarm) and collect a Fog Turbine. Enter Zone 49 or 50 (both Blood Doors) to obtain a Colored Key. Return to the Unsealer Machine in Zone 46-47 to process the cargo (correct machine has illuminated lights on the backside conveyor belt), which stops the Error Alarm. Enter Zone 51 (Class III Mixed Alarm) to retrieve a Bulkhead Key. Insert the key to open Zone 52 and extract with the HISEC Cargo through a Class IV Mixed Alarm that spawns only Chargers and never stops.

### Secondary Objective

Terminal Command: Use the Bulkhead Key in Zone 51H to access Zone 388 (Class III Mixed Alarm). Enter Zone 389 (Class IV Mixed Alarm) and execute the command "RELEASE_CONTAINMENT". This triggers a Room Scan lasting several minutes that opens three Security Doors releasing contained enemies: South room has a Tank and 2 Big Chargers, West room has 6 Hybrids and Cocoons, East room has 5 Big Chargers. Two additional Tanks spawn during the scan. East room proximity triggers respawn mechanics if the team stays 5+ seconds. Collect the Bulkhead Key from Zone 391.

None

## Level Description

Unseal is set in the Mines with infectious fog as a persistent hazard. The level's theme revolves around unsealing: the main objective requires processing HISEC Cargo through an Unsealer Machine, while the secondary objective releases contained enemy specimens via a terminal command. The lore implies Santonian Industries was studying dangerous enemy variants beyond standard specimens. The level is notable for its persistent alarm mechanics, with the final extraction zone featuring a never-ending alarm that continuously spawns Chargers.

## Notable Mechanics

- **Infectious Fog:** Present in both sectors
- **Error Alarm:** Spawns 6 Chargers every ~2:40 until cargo is processed
- **Perpetual Alarm:** Zone 52 extraction alarm never ceases spawning Chargers
- **Room Scan:** Secondary objective triggers multi-minute scan releasing enemies from containment rooms
- **Blood Doors:** Zones 49 and 50
- **Alarm Doors:** Class III Mixed (Zones 51, 388), Class IV Mixed (Zones 48, 52, 389)
- **Unsealer Machine:** Cargo must be processed at the correct machine (identified by illuminated lights)
- **Respawn Mechanics:** West room in secondary triggers respawns based on team proximity duration (5+ seconds)
- **Zone Layout:** Main Zones 45-52; Secondary Zones 388-391
