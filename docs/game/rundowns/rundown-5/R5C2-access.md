# R5C2 - "Access"

## Overview

- **Rundown:** ALT://Rundown 5.0: Rebirth
- **Tier:** C
- **Environment:** Research
- **Dimensions:** No

## Objectives

### Main Objective

Activate Generator Cluster: Collect 5 Power Cells and insert them into a generator system. Retrieve a Bulkhead Key from Zone 211 and unlock access to Zone 215 via a Bulkhead Door Controller in Zone 212. Power Cells are distributed across Zones 213-218. Each cell insertion lowers environmental fog. The fifth insertion triggers a Class IV Alarm. Extract from Zone 220.

### Secondary Objective

Reactor Startup: Access Zone 396 via Bulkhead Key insertion in Zone 211. Initiate a 6-phase reactor startup sequence at the terminal in Zone 397. Each wave has escalating difficulty: Phase 1 (40s code input, Strikers/Shooters), Phase 2 (3m30s, adds Hybrids), Phase 3 (4m30s, Shadows appear), Phase 4 (7m30s, Chargers introduced), Phase 5 (5m, mixed enemies), Phase 6 (5m, Mother spawns at conclusion). Retrieve a colored key from Zone 402 for a code hunt in Zone 401.

### Overload Objective

Terminal Command: Insert a Bulkhead Key in Zone 218 and input the command "OPEN_SECURITY_DOORS" at a terminal in Zone 480. This initiates an extended Room Scan with progressive enemy spawns including Snatchers, Chargers, Giants, Big Chargers, Hybrids, a Tank, and a Mother. Retrieve a Bulkhead Key from Zone 482.

## Level Description

Access is a large-scale expedition set in a Research facility with three full sectors. The main objective revolves around collecting and inserting Power Cells to activate a generator cluster, with each insertion progressively lowering the fog. The secondary objective features a demanding 6-phase reactor startup that escalates through increasingly dangerous enemy types, culminating in a Mother spawn. The overload objective involves a terminal command that triggers an extended Room Scan releasing waves of powerful enemies including a Tank and Mother. The mission name "Access" suggests a central hub controlling sector access and power generation. The overload objective references opening "Zone 100" -- a non-visited location indicating other KSO teams operate adjacent sectors.

## Notable Mechanics

- **Fog Reduction:** Each Power Cell insertion progressively lowers fog levels
- **Alarm Types:** Class III Surge, Class IV Small Sustain, Class IV Mixed, Class V Mixed, Class VI Mixed, Room Scan
- **Blood Doors:** Present in specific zones
- **Reactor Startup:** 6-phase sequence with escalating enemy types (Strikers through Mother)
- **Room Scan:** Extended multi-minute scan with progressive spawns (Overload)
- **Emergency Lockdown:** Security Door protocols active
- **Prisoner Efficiency:** Must complete Secondary and Overload before Main; only 3 Power Cells accessible without Main Bulkhead
- **Zone Layout:** Main Zones 211-220; Secondary Zones 396-403; Overload Zones 480-483 (~20 playable zones total)
- Zone 397 provides extremely high resources (300% ammo, 280% medical, 180% tools)
