# R5D1 - "Even Deeper"

## Overview

- **Rundown:** ALT://Rundown 5.0: Rebirth
- **Tier:** D
- **Environment:** Mines
- **Dimensions:** No

## Objectives

### Main Objective

Reactor Shutdown: Progress from Zone 68 through Zone 69 (Class V Mixed Alarm). Retrieve a Power Cell from Zone 74 or 75 and insert it into Zone 69's Generator to access Zone 70 (Class VI Alarm). Navigate branching paths (76/79 to 77/80 to 78/81) to locate required Colored Keys. Reach Zone 72, collect an optional Bulkhead Key, and initiate the reactor shutdown which triggers an Error Alarm spawning Shadows. Optional: collect a Lore Log in Zone 73 (spawns a Snatcher upon entry). Extract from Zone 68.

### Secondary Objective

Terminal Command: Insert a Bulkhead Key and Door Controller in Zone 72 to access Zone 504. Progress through Zones 505-506 collecting a Colored Key. Retrieve a Power Cell from Zone 508 (Class VI S Alarm) to unlock Zone 507. Enter the command "DEACTIVATE_AIR_FILTRATION" at a terminal in Zone 507 (Class I S Surge Alarm). Optional: retrieve a Fog Turbine from Zone 509. The terminal command causes fog to become infectious and spread to main objective areas.

None

## Level Description

Even Deeper is the first D-tier expedition in Rundown 5, set in the Mines. It features extensive enemy variety with nearly every enemy type in the game present. The level has a branching path design with two separate routes through Zones 74-81, where lighting conditions vary -- Zone 77 has lights on while Zone 80 is pitch black, and the opposite applies to Zones 78/81. The main objective involves shutting down a reactor which triggers Shadow spawns. The secondary objective's terminal command transforms all fog into infectious fog that spreads across sectors. Zone 73 contains a lore log but spawns a Snatcher upon entry. A previous bug caused Zone 507 to occasionally spawn two Big Mothers, which has since been patched.

## Notable Mechanics

- **Branching Paths:** Two separate routes through Zones 74-81 with different lighting conditions
- **Infectious Fog:** Secondary objective transforms all fog to infectious; spreads to main sector
- **Error Alarm:** Reactor shutdown triggers Shadow spawns
- **Alarm Doors:** Class V Mixed (Zone 69), Class VI (Zone 70), Class III Sustain Cluster, Class VI S (Zone 508), Class I S Surge (Zone 507)
- **Darkness Variation:** Zones have alternating lighting (77 lit/80 dark, 78 dark/81 lit)
- **Cocoon Respawn:** Zones 504-505 are Cocoon respawn areas
- **Snatcher Trap:** Zone 73 spawns a Snatcher upon entry (optional lore log zone)
- **Zone Layout:** Main Zones 68-81; Secondary Zones 504-509
- Zone 73 has highest main sector resources (100% ammo, 80% medi, 120% tools); Zone 68 has none
- First level to feature both Spitters and Cocoons simultaneously
