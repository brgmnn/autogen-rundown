# R5D2 - "Error"

## Overview

- **Rundown:** ALT://Rundown 5.0: Rebirth
- **Tier:** D
- **Environment:** Research
- **Dimensions:** No

## Objectives

### Main Objective

Reactor Startup: A single-sector, multi-phase reactor startup sequence. Progress from Zone 280 through Zone 281 (a respawn zone where all enemies respawn indefinitely). Open a Class V Mixed Alarm door to Zone 282 which spawns large quantities of Giants, Hybrids, and Big Shooters. Locate a Power Cell in Zone 284 or 285 and insert it into a generator to unlock Zone 283 (Class X Sustain Cluster Mixed Diminished Alarm). Enter Zone 283 to retrieve a colored keycard while facing Scouts, Mothers, Giants, and Baby Strikers. Navigate to Zone 288 and input "REACTOR_STARTUP" to trigger four code phases: Code 1 (4 min window, Blood Door to Zone 289 with Shadows/Big Shadows), Code 2 (1.5 min window, Zone 290 with Shadows and Giants), Code 3 (45 sec window, Zone 286 with Chargers and Charger Scout), Code 4 (10 min window, three Tanks spawn simultaneously). Extract through Zones 291-293 containing Chargers.

## Level Description

Error is one of the most challenging expeditions in Rundown 5, featuring a single main sector with no secondary or overload objectives. The entire level builds toward a multi-phase reactor startup sequence with progressively tighter code input windows (from 4 minutes down to 45 seconds before expanding to 10 minutes for the final phase). Zone 281 is particularly punishing as all enemies respawn indefinitely. The Class X alarm on Zone 283 is among the most intense alarm encounters in the rundown. The reactor startup culminates with three Tanks spawning simultaneously during the final code phase. C-foam grenades are available in extraction zones to help stall the Tanks.

## Notable Mechanics

- **Infinite Respawn:** Zone 281 enemies respawn indefinitely
- **Class X Alarm:** Zone 283 features a Class X Sustain Cluster Mixed Diminished Alarm
- **Reactor Startup Phases:** Four code phases with decreasing then increasing time windows (4min, 1.5min, 45sec, 10min)
- **Triple Tank Spawn:** Three Tanks spawn simultaneously during Code 4
- **Blood Door:** Zone 289 accessed via Blood Door during Code 1
- **Alarm Doors:** Class V Mixed (Zone 282), Class X Sustain Cluster Mixed Diminished (Zone 283)
- **Known Bug:** Softlock in Zone 281B where players get stuck at bridge stairs base, preventing progression and team scans; requires expedition restart (modded fix available via LGTuner)
- **Dense Fog:** Zone 281B features dense fog
- **Breakable Padlock Door:** Zone 286A to 286B
- **Zone Layout:** Zones 280-295; single sector only
- Zone 288 provides highest resources (280% ammo, 180% medi, 20% tool)
