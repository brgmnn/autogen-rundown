# ALT://Rundown 6.0: Destination

## Overview

- **Release Date:** September 14, 2023
- **Tiers:** A, B, C, D (with X-tier extensions)
- **Total Expeditions:** 13
- **Location:** Various sectors + Destination dimension

## Narrative Summary

The first rundown to introduce alternate dimensions in levels. Prisoners use a Matter Wave Projector to traverse dimensional frequencies and retrieve critical data involving genome databases and PGD encryptions. The plot culminates in discovering "multiple versions of the prisoners," establishing the game's main storyline. Features the Kraken as the rundown boss.

## New Features & Content

- **New Enemy:** Flyer (main new threat of the rundown)
- **New Mechanic:** Matter Wave Projector enables dimensional travel between universe frequencies
- **New Zone:** Destination dimension introduced
- **Restored Weapon:** Scattergun
- **Key Lore:** First confirmation of alternate prisoner versions; beginning of the main game storyline via dimensional mechanics
- **X-tier extensions** explore genome database expansion
- **Tied with Rundown 5.0** for longest rundown (13 expeditions)
