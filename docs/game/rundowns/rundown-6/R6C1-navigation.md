# R6C1 - "Navigation"

## Overview

- **Rundown:** ALT://Rundown 6.0: Destination
- **Tier:** C
- **Environment:** Main Sector
- **Dimensions:** Yes - collecting Data Cubes triggers teleportation to dimensional spaces:
  - First Data Cube (Zone 147): 60-second Flyer survival in Alpha Destination
  - Second Data Cube (Zone 151): 110-second Flyer survival in Alpha Destination
  - Third Data Cube (Zone 153): Brief teleport to Alpha One Installation with unknown creature appearance

## Objectives

### Main Objective

Gather Items: Collect three Data Cubes across multiple zones while managing various alarm types and dimensional encounters.

1. Zone 144 (Class IV Cluster Alarm)
2. Zone 145 - opening triggers Error Alarm spawning Strikers and Hybrids every ~45 seconds
3. Find colored Key in Zone 145
4. Zone 146 - input `DEACTIVATE_ALARMS` at terminal to end Error Alarm
5. Collect first Data Cube in Zone 147 - triggers 60-second Flyer survival
6. Zone 148 (Checkpoint)
7. Zone 149 (Class V Alarm) - contains Giants/Big Shooters
8. Zone 150 has five Scouts
9. Zone 151 (Class II Stealth Scan) - collect second Data Cube - triggers 110-second Flyer survival
10. Zone 152 (Checkpoint)
11. Input `RESET_SECURITY_PROTOCOL` at terminal - initiates Class T Full Team Scan
12. Zone 153 - collect final Data Cube - brief teleport to Alpha One Installation
13. Extract at Zone 154

## Level Description

Navigation is a C-tier gauntlet that tasks players with collecting three Data Cubes, each of which triggers a dimensional teleportation event. The first two cubes send the team to an "Alpha Destination" where they must survive waves of Flyers (60 seconds and 110 seconds respectively). The third cube briefly teleports the team to the Alpha One Installation where an unknown creature appears. Between cube collections, the level features escalating alarm types from Class IV through Class V, including a notable Class T Full Team Scan -- a moving scan that prevents the team from establishing stationary defenses.

Zone 150's five Scouts require careful stealth navigation, and the Error Alarm from Zone 145 persists until manually deactivated via terminal command in Zone 146.

This level has been selected twice for speedrun competitions on the official GTFO Discord -- once by developers and once through community voting.

## Notable Mechanics

- **Class IV Cluster Alarm:** Zone 144 entry.
- **Error Alarm:** Triggered by opening Zone 145, spawns Strikers/Hybrids every ~45 seconds until deactivated via terminal command `DEACTIVATE_ALARMS` in Zone 146.
- **Class V Alarm:** Zone 149.
- **Class II Stealth Scan:** Zone 151.
- **Class T Full Team Scan:** A scan that physically moves around the room as it progresses, preventing stationary defense.
- **Dimensional Teleportation:** Each Data Cube collection triggers a teleport to a Flyer-filled dimension.
- **Scout Navigation:** Zone 150 contains five Scouts requiring stealth.

## Strategy Notes

- Sentries, Bio Tracker, and C-Foam Grenades recommended.
- Manage resources carefully before each Data Cube collection (zones with cubes have 0% resources).
- Zone 152 is heavily stocked (200% ammo/medi) to prepare for the final phases.
- Position sentries strategically for each alarm type.
