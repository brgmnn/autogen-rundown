# R6C2 - "Blind"

## Overview

- **Rundown:** ALT://Rundown 6.0: Destination
- **Tier:** C
- **Environment:** Main Sector + Secondary Sector
- **Dimensions:** No - standard zone progression without dimensional mechanics

## Objectives

### Main Objective

Distribute Powercells: Navigate through multiple zones inserting power cells into generators while managing rising infectious fog levels. Progression involves:

1. Zone 119 (Class IV Alarm)
2. Zone 121 (Blood Door) - insert 1st power cell
3. Execute terminal command in Zone 120 to reset ventilation
4. Insert 2nd power cell in Zone 122
5. Reset ventilation again via terminal
6. Zone 123 (Class V Alarm, Checkpoint)
7. Unlock Bulkhead Door using key
8. Insert final power cell in Zone 125
9. Extract through Zone 126 (Blood Door)

### Secondary Objective

Retrieve Item: Retrieve the HISEC Cargo from Zone 246. Handling it triggers Hybrid enemy spawns every ~45 seconds. Collect a Bulkhead Key from Zone 247 and deliver cargo to the extraction point.

### Overload Objective

None

## Level Description

Blind is a C-tier level centered around the infectious fog mechanic. The core gameplay loop involves inserting power cells into generators, which causes fog levels to rise, then executing terminal commands to reset ventilation and clear the fog. This creates a rhythmic push-and-retreat pattern as players manage infection levels while progressing. Zone 124 features a Mother enemy that is automatically awakened by a Scream Door. Zone 117 is optional but contains high enemy concentration alongside valuable extra resources. The level name likely references the reduced visibility caused by the pervasive fog.

## Notable Mechanics

- **Infectious Fog:** Primary environmental hazard; rises after power cell insertions and can be reset via ventilation terminal commands.
- **Fog Repellers:** Recommended for fog traversal in heavily affected zones.
- **Blood Doors:** Zones 121 and 126 (main sector).
- **Class IV & V Alarms:** Zone 119 (Class IV) and Zone 123 (Class V).
- **Scream Door:** Automatically awakens the Mother in Zone 124.
- **Bulkhead Door:** Separates main and secondary sectors.
- **Checkpoint:** Zone 123.
- **Mother Boss:** Encountered in Zone 124; C-Foam grenades essential for containment.
- **Baby Striker:** Appears in Zone 123.
- **Optional Zone 117:** High enemy concentration but provides 60% ammo, 80% medi, 40% tools.

## Strategy Notes

- All tools have use in this expedition.
- C-Foam grenades are essential for Mother containment.
- Fog Turbines recommended for specific zones.
- Door mining/foaming enables controlled progression.
- Sentry placement is critical for alarm defense.
