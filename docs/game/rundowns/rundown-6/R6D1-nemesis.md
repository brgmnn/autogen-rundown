# R6D1 - "Nemesis"

## Overview

- **Rundown:** ALT://Rundown 6.0: Destination
- **Tier:** D (first D-tier expedition)
- **Environment:** Main Sector
- **Dimensions:** Yes - Matter Wave Projector is inserted into a Jump Gate; Team Scan initiates dimensional jump to the Kraken boss arena (Alpha One)

## Objectives

### Main Objective

Terminal Command + Reactor Startup: Multi-phase expedition culminating in the Kraken boss encounter.

1. Navigate through fog-filled zones (85, 87-88) with standard enemy waves
2. Complete reactor sequence in Zone 90 with four coded input phases:
   - Code 1: 40-second prep
   - Code 2: increasing duration
   - Code 3: further escalation
   - Code 4: 2-minute prep with Tank spawn during final wave
3. Execute Class V Mixed Alarm in Zone 93 to access filtration control
4. Run Class VI Mixed Alarm in Zone 86 (bioscans with fixed patrol patterns)
5. Insert Matter Wave Projector into Jump Gate
6. Trigger final Team Scan to initiate Kraken encounter (3-phase tumor destruction)

### Secondary Objective

None

None

## Level Description

Nemesis is the final story mission of ALT://Rundown 6.0, introducing the Kraken as the final monster of the rundown. The expedition begins in darkness with all lights disabled, progressing through fog-filled zones until the reactor startup sequence activates lighting. The reactor features four waves with escalating difficulty and prep times (40 seconds to 2 minutes). After the reactor, players must clear infectious fog via the terminal command `AIR_FILTRATION_STARTUP` and navigate Class V and Class VI Mixed Alarms. The Class VI Alarm uses fixed scan positions in a predictable lap pattern. The climax involves inserting the Matter Wave Projector into a Jump Gate and teleporting to the Kraken arena for a three-phase boss fight requiring coordinated tumor targeting at distance.

The level has 5 checkpoints -- the highest of any level in the game.

## Notable Mechanics

- **Total Darkness:** All lights initially disabled; activated upon final reactor code input.
- **Infectious Fog:** Present throughout; clearable via `AIR_FILTRATION_STARTUP` terminal command.
- **Fog Turbine:** Optional collectible in Zone 91.
- **Reactor Startup:** Four-phase coded sequence with escalating waves and prep times.
- **Class V Mixed Alarm:** Zone 93.
- **Class VI Mixed Alarm:** Zone 86 with fixed bioscan positions in a predictable lap pattern.
- **Jump Gate:** Matter Wave Projector insertion point for dimensional travel to the Kraken arena.
- **Kraken Boss Fight:** Three-phase tumor destruction at range; Flyers spawn during fight phases.
- **Alarm Skips:** Advanced tactics available ("Reactor Skip," "Alarm Skip") for speedrunning.
- **5 Checkpoints:** Highest checkpoint count in the game.
- **Apex Door:** Door to Jump Gate is not under Emergency Lockdown.

## Strategy Notes

- Minimum two long-range weapons recommended (DMR, Sniper) for the Kraken fight.
- C-Foam Grenades are abundant throughout, making the C-Foam Launcher redundant.
- Sentries recommended for positioning advantage.
- Zone 86 enemies spawn only from the top section during the alarm.
- Kraken arena features caves offering projectile protection during the boss fight.
