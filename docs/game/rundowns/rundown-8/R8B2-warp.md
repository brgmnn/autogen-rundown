# R8B2 - "Warp"

## Overview

- **Rundown:** Rundown 8.0: Duality
- **Tier:** B
- **Environment:** Refinery
- **Dimensions:** Yes -- Multiple dimensional shifts occur throughout both objectives, triggered by terminal commands and security scans. Dimensional travel is central to the level design and story.

## Objectives

### Main Objective

**Terminal Command:** Progress through zones 187-191-188-193. Interact with Bulkhead Door Controller in Zone 187, collect a Bulkhead Key, and execute multiple terminal commands (`BYPASS_ID_Z188`, `FORMAT_SETUP`, `FORMAT_ARTICLE_A`, `FORMAT_ARTICLE_B`, `":_FORM/AT_"`, `EXEC_OVERRIDE`). Navigate through dimensional shifts and extract from Zone 193.

### Secondary Objective

**Terminal Command:** Alternative pathway through zones 326-328-188-193. Follows a similar terminal command sequence but includes a boss encounter with a Kraken at the end instead of standard alarm completion. Choosing the secondary changes the rest of the expedition instead of offering an extra objective. Players completing the secondary still receive credit for the main objective.

### Overload Objective

None

## Level Description

R8B2 introduces a unique mechanic where choosing the secondary objective changes the rest of the expedition rather than adding an extra objective on top. Both paths converge at Zone 188 but the secondary path culminates in a Kraken boss fight. The level features multiple dimensional shifts triggered by terminal commands and security scans. Zone 188 contains multiple terminals, but most are unusable -- players must dimension-hop to locate functional ones.

## Notable Mechanics

- **Mutually exclusive paths** -- secondary replaces main rather than supplementing it; completing secondary still awards main objective credit
- **Multiple dimensional shifts** triggered by terminal commands and security scans
- **Kraken boss fight** at the end of the secondary path
- **Error Alarms** spawning 5 Strikers/Shooters every ~30 seconds during terminal command sequences
- **Unusable terminals** in Zone 188 requiring dimension-hopping to find functional ones
- **Low visibility** in the final alternate dimension alarm section, making Bio-Tracker crucial
- **Flyers** spawn from elevator locations behind extraction scans
- High-damage long-range weapons recommended for the secondary (Kraken fight)
- Funnel-based defense strategies effective for specific alarm spawns
