# R8D1 - "Tandem"

## Overview

- **Rundown:** Rundown 8.0: Duality
- **Tier:** D
- **Environment:** Gardens (mixed industrial/organic)
- **Dimensions:** No

## Objectives

### Main Objective

**Clear a Path:** "Meet Schaeffer in the Gardens." Navigate through 10 zones (420-429) with multiple terminal commands and security door unlocks:

1. Progress from Zone 420 through 421, 422, 423
2. Zone 423 to 424 (Class IV Diminished Alarm)
3. Input `ACCESS_ENTR_Z40` in Zone 424
4. Progress through Zone 426 to 427; input `ROUTE_POWER_Z41`
5. Zone 428: input `JUNCTION_1_Z425`; retrieve Fog Turbine
6. Return to Zone 424; input `ADMIN_OVRD_Z429`
7. Zone 429: input `JUNCTION_2_Z425`
8. Zone 425: complete Team Scan
9. Extract at Zone 420

**Achievement:** "Pure Will" requires completing the expedition without using Medical Packs or Disinfection Stations.

None

None

## Level Description

R8D1 is the first D-tier expedition in Rundown 8, tasking players with meeting Schaeffer in the Gardens. The level is notable for being the first expedition to feature every environmental hazard in the game. It involves navigating through 10 zones with multiple terminal command sequences and strategic fog management. Zones 420-422 and 424 feature respawning cocoons, adding persistent threat throughout the expedition.

## Notable Mechanics

- **First expedition to feature every environmental hazard in the game:** Infectious Fog, Spitters, and Corrupt Scans
- **Corrupt Scans** appear in fixed locations; only the initial one before Zone 422 is required and non-respawning
- **Respawning cocoons** in Zones 420, 421, 422, and 424
- **Escalating Infectious Fog** in Zone 424 requiring upward escape
- **Fog Turbine** retrieval in Zone 428
- **Alarm types:** Class IV Diminished, Class III, Team Scan, Error Alarm, Room Scan, Surge Alarm
- **Big Chargers** in Zone 424 spawn only when doors open and respawn exactly once via terminal command
- Terminal commands: `ACCESS_ENTR_Z40`, `ROUTE_POWER_Z41`, `JUNCTION_1_Z425`, `ADMIN_OVRD_Z429`, `JUNCTION_2_Z425`
- Tight resource distribution: many zones have 0% in one or more categories
